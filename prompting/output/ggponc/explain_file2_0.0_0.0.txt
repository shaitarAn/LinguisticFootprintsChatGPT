Der Text beschreibt verschiedene Urinmarker, die zur Diagnose von Blasenkarzinomen eingesetzt werden können. Es wird erwähnt, dass das Protein NMP22 in malignen Urothelzellen häufiger vorkommt als in normalen Zellen und dass Patienten mit Blasenkarzinom vermehrt NMP22 im Urin ausscheiden. Es wird darauf hingewiesen, dass der Test auf NMP22 unspezifisch für ein Urothelkarzinom ist, da er apoptotische Zellbestandteile im Urin nachweist. Es wird auch erwähnt, dass sowohl der quantitative als auch der qualitative NMP22-Test von der FDA für die Nachsorge bei nicht-muskelinvasiven Blasenkarzinomen zugelassen sind.

Es werden verschiedene Studien zitiert, die die Sensitivität und Spezifität des NMP22-Tests untersucht haben. Es wird festgestellt, dass die Sensitivität des Tests mit der Anzahl der Tumore korreliert und dass der Test in einigen Studien eine höhere Sensitivität als die Urinzytologie aufweist. Es wird jedoch auch darauf hingewiesen, dass der NMP22-Test eine hohe falsch positive Rate aufweist und dass es eine große Variabilität in den Testergebnissen gibt.

Es werden auch andere Urinmarker wie das Bladder Tumour Antigen (BTA), Survivin, CYFRA 21-1, Telomerase, MCM Proteine und verschiedene Genanalysen erwähnt. Es wird festgestellt, dass keiner dieser Marker bisher für eine einfache praktische Anwendung entwickelt wurde und dass die Datenlage für eine klinische Anwendung nicht ausreichend ist.

Es wird auch auf die Probleme und Einschränkungen der verschiedenen Urinmarker hingewiesen, wie z.B. falsch positive Ergebnisse bei Hämaturie oder anderen urologischen Erkrankungen. Es wird betont, dass eine Kombination mehrerer Urinmarker oder die Selektion von Risikopatienten anhand klinischer Parameter die diagnostische Genauigkeit verbessern kann.

Insgesamt zeigt der Text, dass es verschiedene Urinmarker gibt, die zur Diagnose von Blasenkarzinomen eingesetzt werden können, aber dass keine der Methoden bisher als alleiniger diagnostischer Test ausreichend validiert ist. Weitere Forschung ist erforderlich, um die Genauigkeit und klinische Anwendung dieser Marker zu bestimmen.
