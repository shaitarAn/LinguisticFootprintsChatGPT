In dem Text geht es um die präoperative biliäre Drainage (PBD) bei Patienten mit Gallengangskarzinomen. Es wird darauf hingewiesen, dass die Frage, ob eine PBD durchgeführt werden sollte, noch nicht endgültig geklärt ist. Die vorhandene Evidenz basiert hauptsächlich auf retrospektiven Analysen und nur vereinzelt auf prospektiven randomisierten Studien.

Es wird auf eine kürzlich veröffentlichte Studie verwiesen, die gezeigt hat, dass eine Hyperbilirubinämie (erhöhte Bilirubinspiegel im Blut) ein wichtiger modifizierbarer Risikofaktor für ein negatives postoperatives Outcome ist. Es gibt jedoch keine ausreichenden Beweise für einen bestimmten Grenzwert der Bilirubinspiegel.

Es wird auch darauf hingewiesen, dass die PBD selbst das postoperative Outcome negativ beeinflussen kann, zum Beispiel durch Komplikationen der interventionellen Therapie. Es wird vermutet, dass eine erhöhte Morbidität auch bei erfolgreicher Drainage auf das Einschleppen von Keimen in das Gallenwegsystem während der interventionellen Therapie zurückzuführen ist.

Es wird betont, dass eine PBD unstrittig notwendig ist, wenn eine Cholangitis (Entzündung der Gallengänge) vorliegt. Die Indikation zur PBD muss jedoch für das intrahepatische Gallengangskarzinom und das perihiläre Gallengangskarzinom getrennt betrachtet werden. Bei intrahepatischen Gallengangskarzinomen, die selten mit einer Kompression der zentralen Gallenwegsstrukturen und einem Ikterus einhergehen, ist eine präoperative Gallengangsdekompression nur vor ausgedehnten Resektionen und Beteiligung der Gallenwege des "Future Liver Remnant" erforderlich.

Es wird auch darauf hingewiesen, dass die optimale Technik für die präoperative Drainage noch nicht abschließend geklärt ist. Es gibt widersprüchliche Ergebnisse hinsichtlich Komplikationen und Erfolgsraten beim endoskopischen oder perkutanen Vorgehen. Eine Metaanalyse hat gezeigt, dass die Morbidität nach endoskopischer Drainage vergleichbar oder etwas höher ist als nach perkutaner Drainage.

Es wird auf eine randomisierte Studie verwiesen, die vorzeitig abgebrochen wurde, da in der PTCD-Gruppe eine signifikant höhere Gesamtmortalität und eine höhere Rate an Cholangitis im Vergleich zur endoskopischen Drainage-Gruppe festgestellt wurde. Es wird jedoch darauf hingewiesen, dass bei 56% der Patienten, die eine endoskopische Drainage hatten, zusätzlich eine perkutane Drainage erforderlich war.
